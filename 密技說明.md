# 密技說明

本專案有四個密技功能，可以在首頁任何頁面使用。

---

## 1. money 密技

### 觸發方式
在首頁任何地方輸入 `money`（不在輸入框內）

### 功能
- 本地星星儲值
- 解鎖關卡並通關（Google Sheet）
- 解鎖對應卡片（Google Sheet）

### 使用步驟

#### 星星儲值
1. 輸入 `money` 打開管理者操作視窗
2. 選擇操作類型「儲值星星」
3. 輸入管理者密碼
4. 輸入玩家 ID 或名稱
5. 輸入星星數量
6. 輸入原因
7. 確認送出 → 本地星星增加

#### 解鎖關卡
1. 輸入 `money` 打開管理者操作視窗
2. 選擇操作類型「解鎖關卡」
3. 輸入管理者密碼
4. 輸入玩家 ID 或名稱
5. 從關卡選單選擇要解鎖的關卡（43 個關卡：20 星座 + 23 希臘神祇）
6. 輸入原因
7. 確認送出 → Google Sheet 寫入 `unlockedLevels` 和 `unlockedCards`

### 關卡列表

#### 星座關卡（20個）
- 白羊角、金牛座、雙子座、巨蟹座、獅子座、處女座、天秤座、天蠍座、射手座、摩羯座、水瓶座、雙魚座
- 仙女座、天鵝座、獵戶座、飛馬座、仙后座、天蠍座、鳳凰座、船帆座

#### 希臘神祇關卡（23個）
- 宙斯、赫拉、波塞冬、雅典娜、阿波羅、阿爾忒彌斯、阿瑞斯、阿佛洛狄忒、哈迪斯、得墨忒耳、狄俄尼索斯、赫淮斯托斯、赫爾墨斯、赫斯提亞、珀耳塞福涅、普羅米修斯、瑞亞、蓋亞、克洛諾斯、阿特拉斯、尼刻、厄羅斯

### 注意事項
- 星星儲值只影響本地 `totalStars`
- 解鎖關卡會寫入 Google Sheet，玩家重新整理後同步
- 解鎖關卡會同時解鎖對應卡片（如果有對應）
- 解鎖關卡會自動標記為通關（加入 `passed_atlas`）

## 2. english 密技

### 觸發方式
在首頁任何地方輸入 `english`（不在輸入框內）

### 功能
- 本地解鎖關卡並通關（支援批量）
- 本地解鎖對應卡片（支援批量）

### 使用步驟
1. 輸入 `english` 打開解鎖關卡視窗
2. 從關卡選單選擇要解鎖的關卡（43 個關卡，可多選）
3. 輸入原因
4. 確認送出 → 本地 `passed_atlas` 和 `ownedCards` 更新

### 注意事項
- 只影響本地 localStorage
- 換電腦登入後不會同步
- 支援批量選擇關卡（按住 Ctrl/Cmd 多選）
- 解鎖關卡會同時解鎖對應卡片（如果有對應）
- 解鎖關卡會自動標記為通關
- 已解鎖的關卡不會重複計算

## 3. card 密技

### 觸發方式
在首頁任何地方輸入 `card`（不在輸入框內）

### 功能
- 本地解鎖卡片（支援批量）

### 使用步驟
1. 輸入 `card` 打開卡片解鎖視窗
2. 從卡片選單選擇要解鎖的卡片（依系列分組，可多選）
3. 查看卡片預覽（圖片、名稱、稀有度、系列、ID）
4. 輸入原因（選填）
5. 確認送出 → 本地 `ownedCards` 更新

### 注意事項
- 只影響本地 localStorage
- 換電腦登入後不會同步
- 支援批量選擇卡片（按住 Ctrl/Cmd 多選）
- 卡片 ID 使用 `card.word`，與卡片倉庫判斷邏輯一致
- 解鎖後卡片倉庫會立即顯示已解鎖
- 已解鎖的卡片不會重複計算

---

## 4. name 密技

### 觸發方式
在首頁任何地方輸入 `name`（不在輸入框內）

### 功能
- 修改玩家名稱

### 使用步驟
1. 輸入 `name` 打開修改名字視窗
2. 輸入新名字（最多10個字）
3. 確認送出 → 本地 `playerName` 更新

### 注意事項
- 只影響本地 localStorage
- 換電腦登入後不會同步
- 會同步更新 LinkageSystem 和頁面顯示
- 名稱限制最多10個字

## 技術說明

### 本地儲存
- `totalStars` - 星星數量
- `passed_atlas` - 已通關關卡陣列
- `ownedCards` - 已解鎖卡片物件（key 為 card.word）

### Google Sheet
- `unlockedLevels` - 已解鎖關卡（逗號分隔）
- `unlockedCards` - 已解鎖卡片（逗號分隔）
- `updatedAt` - 最後更新時間

### 後端 API
- `adminUnlockItem` - 解鎖項目（關卡、卡片等）
- `adminAdjustStars` - 調整星星數量（目前未使用，改為本地）
